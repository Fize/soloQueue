2. PROJECT PURPOSE & FEATURES
───────────────────────────────────────────────────────────────────────────────

Project: SoloQueue - AI Multi-Agent Collaboration Desktop Application

Vision:
  ✅ Actor-based distributed system (each agent = independent actor)
  ✅ Asynchronous message-driven communication
  ✅ XState v5 for deterministic state management
  ✅ SQLite persistence with automatic state snapshots
  ✅ Streaming LLM support (DeepSeek, OpenAI)
  ✅ Supervision trees for fault tolerance
  ✅ Desktop application via Tauri
  ✅ Type-safe implementation (full TypeScript + Zod)

Core Use Case:
  1. User creates Team with multiple Agents
  2. User initiates conversation (Leader Agent receives message)
  3. Leader Agent processes and can delegate to sub-agents
  4. Sub-agents work concurrently, return results
  5. All state automatically persisted to SQLite

8. ARCHITECTURE DIAGRAMS
───────────────────────────────────────────────────────────────────────────────

ACTOR SYSTEM ARCHITECTURE:
  ┌─────────────────────────────────────┐
  │        ActorSystem Registry         │
  │  [Leader] [Worker1] [Worker2] [...] │
  └──────────────┬──────────────────────┘
                 │
         ┌───────┴──────┐
         ▼              ▼
    Supervisor      Router
  (Recovery)    (Load Balance)
         │              │
         ▼              ▼
    [Actors recover]  [Messages routed]

MESSAGE FLOW:
  Frontend → Tauri IPC → ActorSystem.dispatch()
                         ↓
                    Router → Target Actor
                         ↓
                    State Machine (FSM)
                         ↓
                    [LLM | Delegate | Respond]
                         ↓
                    Persist → SQLite/Logs

STATE MACHINE (Agent):
  idle ──task──> processing ──┬──> waitingLLM ──┐
                               │                  │
                               ├──> delegating ──┤
                               │                  │
                               └──> responding ───┘
                                     ↓
                                  [done]
                                     ↓
                                    idle
