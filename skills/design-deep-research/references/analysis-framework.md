# 风格分析框架

设计调研中，从素材到审美规律的提炼方法论。

---

## 一、素材聚类方法论

### 聚类原则

将所有采集到的素材按视觉风格聚类，形成 3-5 个设计方向：

1. **先标签化**：每个素材按视觉特征打标签（色彩倾向、图形风格、排版方式、材质特征、情绪调性）
2. **标签聚类**：根据标签重叠度，将素材分为若干候选聚类
3. **合并精炼**：合并小聚类、拆分大聚类，确保每类有 ≥3 个案例且方向间有明显区分度
4. **命名**：每个方向取一个有辨识度的风格名称

### 区分度标准

- 两个方向在 ≥2 个视觉维度上有明显差异（如色彩+图形不同，而非仅色彩微调）
- 每个方向能用一句话说清"这是什么风格"
- 方向间不能是"浅色版 vs 深色版"这种单一维度的渐变

## 二、单类风格的审美分析维度

对每个设计方向，按以下 7 个维度进行独立分析：

### 2.1 色彩

| 子维度 | 分析内容 |
|--------|---------|
| 主色调 | 识别主色及色相范围（如暖橙、冷蓝紫） |
| 辅助色 | 辅助色数量、色相、作用（强调/平衡/过渡） |
| 配色逻辑 | 对比/渐变/邻近/互补/单色/分裂互补 |
| 饱和度 | 高饱和/低饱和/混灰 |
| 明度 | 高亮/暗调/中间调 |
| 色彩情绪 | 色彩传递的心理感受 |

### 2.2 排版

| 子维度 | 分析内容 |
|--------|---------|
| 字体风格 | 衬线/无衬线/手写/装饰/字重选择 |
| 层级结构 | 标题-副标题-正文的比例关系、强调方式 |
| 留白模式 | 密集/宽松、对称/不对称留白 |
| 对齐方式 | 左对齐/居中/右对齐/自由排列 |
| 文字排版 | 行距/字距/行长控制 |
| 排版节奏 | 信息分组方式、阅读引导路径 |

### 2.3 图形

| 子维度 | 分析内容 |
|--------|---------|
| 图形语言 | 几何/有机/手绘/3D/像素/抽象 |
| 图标风格 | 线性/面性/扁平/3D/动态 |
| 插画风格 | 扁平/渐变/噪点/水彩/矢量/拼贴 |
| 图形密度 | 简约/复杂/繁复 |
| 图形来源 | 原创/素材/摄影转图形 |

### 2.4 材质与光影

| 子维度 | 分析内容 |
|--------|---------|
| 纹理 | 纸张/布料/金属/石材/木纹/颗粒/噪点 |
| 渐变 | 线性/径向/自由/网格渐变、渐变幅度 |
| 阴影 | 投影/内阴影/长阴影/柔和/硬朗 |
| 发光 | 霓虹/背光/光晕/辉光 |
| 玻璃态 | 毛玻璃/磨砂/透明/反射 |
| 立体感 | 2D/2.5D/3D/拟物/扁平 |

### 2.5 构图

| 子维度 | 分析内容 |
|--------|---------|
| 布局结构 | 居中/对称/黄金分割/网格/自由/满版 |
| 视觉重心 | 中心/偏上/偏下/对角/散点 |
| 比例关系 | 图文比、正负空间比 |
| 破格手法 | 出血/叠压/出框/留白破格 |
| 视线引导 | 阅读路径设计、视觉动线 |

### 2.6 情绪

| 子维度 | 分析内容 |
|--------|---------|
| 调性关键词 | 3-5个情绪词（如：温暖/克制/理性/诗意/先锋） |
| 目标感受 | 设计想传达给观众的心理感受 |
| 适用氛围 | 什么场景/场合下使用这个风格最合适 |

### 2.7 典型手法

3-5 个该方向常用的具体设计技巧（可操作，非抽象描述）。例如：
- "用大号无衬线字体作为视觉主体，文字即图形"
- "在摄影背景上叠加半透明几何形状"
- "通过极端留白制造高级感"

## 三、Prompt 推导规则

### 3.1 Prompt 结构模板

对每个设计方向，基于其视觉特征推导适配 nano banana / image2 的生图 Prompt：

```
[主体/主题], [风格/流派], [色彩描述], [材质/光影], [构图/视角], [氛围/情绪], --ar [宽:高] --style [选项]
```

### 3.2 Prompt 要素推导

| Prompt 要素 | 从分析维度映射 |
|-------------|--------------|
| 主体/主题 | 设计类型 + 行业领域 |
| 风格/流派 | 图形语言 + 排版风格 |
| 色彩描述 | 主色调 + 配色逻辑 |
| 材质/光影 | 纹理 + 光影 + 渐变 |
| 构图/视角 | 布局结构 + 视觉重心 |
| 氛围/情绪 | 情绪调性关键词 |

### 3.3 Prompt 生成原则

1. **基于实际调研**：Prompt 中的视觉要素必须来自实际调研到的素材，不能编造
2. **考虑参考图锚点**：如果用户提供了参考图，Prompt 中需要体现参考图的风格锚点
3. **可调节参数**：用 `[占位符]` 标注用户需要替换的部分（如 `[产品名称]`、`[品牌色]`）
4. **说明调节方向**：每个参数标注"增大 → 更[XX]，减小 → 更[XX]"
5. **风格强度控制**：用形容词程度调节（如 "slightly"、"very"、"extremely"）

### 3.4 Prompt 示例格式

```prompt
[产品主体]，[风格名称]风格，[主色调]为主[配色逻辑]配色，[材质描述]，[构图方式]，[情绪氛围]，--ar 16:9 --style [选项]
```

每个 Prompt 附带调节说明：
> - 替换 `[产品主体]` 为你的设计对象
> - 调整 `[风格名称]` 的形容词强度可控制风格偏向
> - `--ar` 可根据需要调整宽高比
