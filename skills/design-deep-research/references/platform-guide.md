# 设计调研平台指南

平面设计素材与灵感调研的核心平台清单，按用途分类。

---

## 一、国际灵感与创意趋势

### Behance (behance.net)

- **特点**：Adobe 旗下全球顶级设计师社区。国际化视野广，能看到全球最前沿的品牌全案、VI 视觉系统、版式和包装设计。
- **搜索策略**：英文关键词，关注 Case Study。关键词组合：[design type] + [style] + [industry]
- **素材类型**：完整项目展示，含设计过程、应用效果
- **搜索技巧**：使用 Behance 的 "Curated Galleries" 和 "Moodboards" 功能

### Pinterest (pinterest.com)

- **特点**：瀑布流推荐算法极其精准，点开一张图会持续推送类似风格的版式、配色和字体设计。做情绪板（Moodboard）的最佳平台。
- **搜索策略**：英文关键词，利用算法发散。先搜一个精准词，然后顺着推荐流收集
- **素材类型**：单张灵感图、情绪板
- **搜索技巧**：找到高质量 Pin 后，查看该 Pin 的 "More like this"

### Dribbble (dribbble.com)

- **特点**：偏向 UI/UX 和动效，但插画风格、色彩搭配、极简版式对平面设计师是极好养料，多是精致概念稿。
- **搜索策略**：英文关键词，关注 "Popular" 和 "Goods" 分类
- **素材类型**：单张概念稿、局部细节

## 二、高质量 Mockup（样机/包装/贴图）

### Mockup World (mockupworld.co)

- **特点**：全球最大免费样机聚合网站。iPhone、包装盒、纸袋、户外广告牌、书籍画册应有尽有，全部免费 PSD。
- **搜索策略**：按品类搜索样机类型
- **素材类型**：PSD 格式样机文件
- **标注**：免费可商用

### Graphic Burger (graphicburger.com)

- **特点**：老牌优质设计资源站，Mockup 质量极高，纹理细节（纸张质感、烫金效果）非常逼真。同时提供纹理背景和字体。
- **搜索策略**：按资源类型搜索
- **素材类型**：Mockup、纹理背景、字体
- **标注**：免费+付费混合，注意授权

### Yellow Images (yellowimages.com)

- **特点**：3D 样机和包装贴图的终极选择。独家 3D 样机可任意旋转角度、更换材质，适合高端快消品包装设计。
- **搜索策略**：按品类名称搜索
- **素材类型**：3D 样机（PSD/Smart Object）
- **标注**：付费为主

## 三、矢量、插画与设计组件

### Freepik (freepik.com)

- **特点**：极其庞大的素材库，海量矢量图（AI/EPS）、插画、背景纹理、海报模板。每天有免费下载额度。
- **搜索策略**：中英文关键词均可，标注免费/商用筛选
- **素材类型**：矢量图、插画、模板、背景
- **标注**：免费用户每天有限额，商业用途需确认授权

### Vecteezy (vecteezy.com)

- **特点**：专注矢量图形。找特定风格的边框、Icon、背景或装饰图案时效率高。
- **搜索策略**：英文关键词，格式筛选（AI/EPS/SVG）
- **素材类型**：矢量图形、Icon、边框、装饰图案

## 四、免版权高清图库

### Unsplash (unsplash.com)

- **特点**：照片极具故事感和文艺调性，光影和构图远超传统商用图库。适合海报背景、主视觉配图。
- **搜索策略**：英文关键词
- **素材类型**：高清摄影照片
- **标注**：CC0 免版权可商用

### Pexels / Pixabay (pexels.com / pixabay.com)

- **特点**：已被 Canva 收购，图库非常全。除高清图片外还有 4K 视频素材。中文搜索支持优于 Unsplash。
- **搜索策略**：中英文关键词均可
- **素材类型**：高清图片、4K 视频
- **标注**：CC0 免版权可商用

## 五、国内综合与垂直平台

### 站酷 (zcool.com.cn)

- **特点**：国内最大综合设计社区，涵盖平面/UI/插画/包装等全品类。
- **搜索策略**：中文关键词
- **素材类型**：完整项目展示

### 花瓣 (huaban.com)

- **特点**：国内版 Pinterest，瀑布流灵感采集平台。
- **搜索策略**：中文关键词，利用推荐流发散
- **素材类型**：单张灵感图、画板

### 古田路9号 (gtn9.com)

- **特点**：垂直品牌设计、VI 视觉、包装设计的专业平台。看国内落地商业全案的最佳平台。
- **搜索策略**：中文关键词，关注商业全案
- **素材类型**：完整品牌全案、包装设计

### 大作网 (bigbigwork.com)

- **特点**：聚合海内外几十个设计网站的搜索引擎，支持"一键看全球"，找灵感效率极高。
- **搜索策略**：中英文关键词均可
- **素材类型**：聚合搜索结果

### 优设网 (uisdc.com)

- **特点**：不仅有素材，还有海量免费平面设计教程、字体版权检测工具和行业干货。
- **搜索策略**：中文关键词
- **素材类型**：教程、素材、工具

## 平台选择策略

根据需求类型，优先推荐的平台组合：

| 需求类型    | 推荐平台                                           |
| ----------- | -------------------------------------------------- |
| 品牌全案/VI | Behance + 古田路9号 + Pinterest                    |
| 包装设计    | Behance + Yellow Images + Mockup World + 古田路9号 |
| 海报/版式   | Pinterest + Dribbble + Unsplash                    |
| 插画风格    | Dribbble + Freepik + Behance                       |
| 综合找灵感  | Pinterest + 大作网 + 站酷                          |
| 落地素材    | Freepik + Unsplash + Pexels                        |
| 样机展示    | Mockup World + Graphic Burger + Yellow Images      |

## 信息源排除规则

- 不使用百度图片（质量低、版权不清）
- 不使用微信公众号文章（封闭生态、无法验证）
- 不使用知乎作为设计素材来源（非专业设计平台）
