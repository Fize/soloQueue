# 融合笔法通用执行规范

> 本文件定义"沉默见证者"叙事笔法。适用于以人物关系与命运交织为核心驱动力的虚构叙事作品，核心适配场景为羁绊驱动型群像故事，不适用于纯粹的力量升级流或生存主义题材。
>
> 本规范是 `writing-style.md` 的特化补充，在《天元鉴》项目中为**强制文风基线**。

---

## 一、总纲

**沉默见证者。** 以冷静克制的旁观视角凝视人物，不解释心理，不评判选择，只记录可见的动作、对话与细节。情感藏于细节，逻辑融于动作，爆发落于瞬间。

---

## 二、核心原则

**克制。** 不写尽——情感不直述，用外部细节折射；战斗不浮夸，用动作与结果说话；高潮不嘶喊，用一句对白收尾。让读者在留白中自己补全暗涌。

---

## 三、叙述姿态

始终维持同一距离。不对角色的愤怒做内心剖析，只写指节发白。不对角色的痛苦做长段回忆，只写某个重复动作比平时更慢。不对角色的决意做铺垫，只写他站在某个地方的时辰。

### 3.1 内心独白禁令（红线 · 零容忍）

**禁止出现**以下心理活动标记词：

| 禁用标记 | 替代方向                         |
| -------- | -------------------------------- |
| 他想     | 转为外部动作或细节               |
| 他感到   | 转为身体反应（指节、呼吸、眼神） |
| 他意识到 | 转为动作突变或停顿               |
| 他回忆起 | 转为物品触发或场景触动           |

心理活动一律转化为外部细节。违规一次即为红线 FAIL。

## 四、场景执行规范

### 4.1 日常与情感线

| 要点     | 规范                                                     |
| -------- | -------------------------------------------------------- |
| 载体     | 以具体细节承载情感，不直接描述情感状态                   |
| 手法     | 使用重复动作在不同场景中的变化来推进关系                 |
| 判断标准 | 不写"他们关系好"，写"东西递过去时对方还没开口就已经接了" |
| 典型场景 | 篝火沉默、分食、推了半寸的物件、放在某处不拿走的物品     |

### 4.2 策略型战斗

| 要点     | 规范                                                                          |
| -------- | ----------------------------------------------------------------------------- |
| 原则     | 写实为底，每次出手需交代逻辑                                                  |
| 必写项   | 力量如何运转、功法相克关系、境界差距体现、阵型配合过程                        |
| 群战要求 | 明确多人物定位（主攻/次攻/控场/远程），配合靠动作先后与补位时机体现，不在言语 |
| 判断标准 | 读者能清晰复盘整场战斗的逻辑链                                                |

### 4.3 爆发型战斗

| 要点     | 规范                           |
| -------- | ------------------------------ |
| 适用场景 | 情感驱动的决胜一击，非策略博弈 |
| 写法     | 不写力量参数，只写动作与结果   |
| 切换标志 | 这一击出去，不能回头看         |
| 收尾     | 打完即止，不拖泥带水           |

### 4.4 高潮对白

| 要点       | 规范                                                                                                                       |
| ---------- | -------------------------------------------------------------------------------------------------------------------------- |
| 原则       | 必须是极度符合人物身份与立场交锋、掷地有声的话，绝非无意义的说教或设定说明。                                               |
| 比例与篇幅 | 允许角色在情感爆发或意志决裂时说出具张力的大段对白（如宣示立场、斩断关系、绝交饮酒），不必强求字数极简，但严禁废话与注水。 |
| 要求       | 不解释，不展开，一锤定音，且话音落地即直接产生动作或局势质变。                                                             |
| 特殊处理   | 沉默也可以是对白——在不拔剑、不回答的静默中形成张力对峙。                                                                   |

### 4.5 情感收尾

| 要点     | 规范                                               |
| -------- | -------------------------------------------------- |
| 时机     | 高潮战斗或情感爆发后必须跟随静场                   |
| 写法     | 用安静的具体动作释放余韵，不庆祝、不拥抱、不流泪   |
| 典型动作 | 坐在地上看自己的手、把某物放在某处、说一句无关的话 |

## 五、叙事技法

### 5.1 伏笔与悬念

| 要点 | 规范                                                                     |
| ---- | ------------------------------------------------------------------------ |
| 伏笔 | 藏于动作或物品，出现时只是场景一部分，不做强调                           |
| 悬念 | 不靠叙述者刻意隐瞒，靠角色视角的天然盲区——旁观者不知道的事，读者也不知道 |

### 5.2 时间处理

| 要点     | 规范                                       |
| -------- | ------------------------------------------ |
| 主时序   | 顺时序推进，让因果在时间中自然呈现         |
| 预叙     | 仅用于全书开篇一句定调，不在正文中频繁使用 |
| 章节钩子 | 仅用于关键章节末尾，以动作或对白戛然而止   |

### 5.3 叙事节奏

| 要点     | 规范                                                   |
| -------- | ------------------------------------------------------ |
| 原则     | 张弛交替，高潮后必须跟随静场                           |
| 交替模式 | 大战后接沉默场景，弑杀后接漫长逃亡，紧张与舒缓交替推进 |

## 六、通用自检清单

| 序号 | 检查项                                                                                                                                                                |
| ---- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1    | 是否写了一句内心独白？→ 删掉，改为外部动作或细节                                                                                                                      |
| 2    | 战斗是否每一招都有明确的结果和逻辑？→ 否则补清力量流转或克制链                                                                                                        |
| 3    | 情感爆发后是否立刻接安静场景？→ 否则插入余韵                                                                                                                          |
| 4    | 高潮对白是否流于说教或软弱无力？→ 删去解释或灌水词，提炼为最符合人物风骨与交锋张力的狠话/真话。                                                                       |
| 5    | 是否解释了角色为什么做某件事？→ 删掉解释，只留动作                                                                                                                    |
| 6    | 同一段落是否混用了细腻铺陈与极简金句？→ 场景内保持笔法统一                                                                                                            |
| 7    | 整章对话占比是否过低？（建议 20%-35%，高潮交锋章可适当增加）→ 若通篇全为场景/动作物理测绘，应当通过人物的对峙、机锋或侧面谈论进行丰富，杜绝变成冰冷干瘪的画面流水账。 |

---

## 七、一句话记忆法则

**让读者看见动作，听见对白，感受留白。你只负责记录，不负责解释。解释是读者的事。**
