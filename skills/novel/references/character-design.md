# 人物设计（通用 · 金庸人物优先原则）

> 主角 / 配角 / 反派的设计标准。拒绝脸谱化、拒绝金手指、拒绝工具人。
> 人物设计必须联动 `jin-yong-writing-perspective` 技能的 7 个核心心智模型执行——尤其是模型1（人物驱动情节）、模型5（道德灰度）、模型6（间接呈现）。
> 文风相关见 [writing-style.md](writing-style.md)；对话情绪纹理见 [writing-style.md](writing-style.md) §五。

---

## 〇、金庸人物核心原则（强制绑定 jin-yong-writing-perspective）

### 0.1 人物驱动情节（模型1）

**写作逻辑**：先构思人物（性格、欲望、恐惧、矛盾），再配故事。情节和人物冲突时——**改情节**。

**决策问题**（每次推演人物线时必须问）：
- 以这个人物的性格，ta在这个处境下会怎么做？如果答案和你的情节设计不一致，情节需要改。
- 这个情节是否出自人物的性格？还是作者为了推进剧情"安排"的？
- 人物在这场戏中做了选择吗？这个选择是否让读者更了解ta？

**违规示例**：为了制造冲突让善良的主角突然冷漠、让谨慎的主角突然鲁莽——这是为情节牺牲人物。

### 0.2 道德灰度（模型5）

**写作逻辑**：没有纯黑纯白。每个正派有让人皱眉的缺点，每个反派有令人心软的瞬间。

**设计原则**：
- 写反派时：给ta一个"如果换个处境，可能是好人"的瞬间。可恨之人必有可怜之处。
- 写正派时：给ta一个让人不舒服的缺点或选择。
- 最精彩的是"正邪转换"——让读者看到一个好人如何一步步走向黑暗（或反之）。

**决策问题**：
- 这个人物有没有让读者产生"复杂情感"的能力？不是单纯的爱或恨。
- 这个人物如果只有一面（纯善/纯恶），是否应该重写？

### 0.3 间接呈现（模型6 · 技法4·一击两鸣）

**写作逻辑**：写一个人物最厉害的地方，不是直接描述ta，而是写ta对其他人产生了什么影响。

**间接呈现手法见 [writing-style.md](writing-style.md) §1.1。

## 一、主角设计（三要素 + 非全能性）

### 1.1 三要素（缺一不可）

```
欲望（想要什么） + 恐惧（害怕什么） + 矛盾（内心挣扎什么）
```

三者必须同时具备，缺一则人物扁平。

### 1.2 好主角示范

| 主角 | 欲望 | 恐惧 | 矛盾 |
|---|---|---|---|---|
| 《射雕》郭靖 | 侠之大者，为国为民 | 辜负期望 | 愚钝与坚韧的并存 |
| 《天龙》萧峰 | 查明身世、守护兄弟 | 自己成为自己最憎恨的人 | 契丹血统 vs 宋人养育 |
| 《神雕》杨过 | 与姑姑团聚 | 被世间抛弃 | 叛逆不羁 vs 至情至性 |
| 《笑傲》令狐冲 | 自由自在 | 被规则束缚 | 向往自由 vs 枷锁缠身 |
| 《悟空传》孙悟空 | 自由 | 被操控 | 斗战胜佛 vs 齐天大圣 |
| 《庆余年》范闲 | 守护想守护的人 | 再失去一次 | 现代灵魂 vs 古代秩序 |
| 《将夜》宁缺 | 活下来、报仇 | 被天命裁决 | 凡人体质 vs 永夜之主 |

### 1.3 坏主角（严禁）

- **完美无缺型**：没有弱点、没有恐惧、没有挣扎
- **纯金手指型**：一切成就靠开挂，不靠选择与代价
- **目的单一型**：除了升级打怪，对世界毫无好奇、对关系毫无感知
- **圣母型**：所有人都喜欢他，所有矛盾都能化解

### 1.4 非全能性（强制）

- 主角必须有**专精外的难题**会触发"求援机制"
- 遇到自己不擅长的领域，必须选择：求助 / 妥协 / 付出代价 / 失败
- 禁止"主角什么都会、什么都最强"

## 二、配角设计（四问）

### 2.1 强制四问

每个配角出场前，必须回答：

1. **他 / 她想要什么？**（独立欲望，不能只是"帮主角"）
2. **他 / 她害怕失去什么？**（独立恐惧）
3. **他 / 她为什么帮助 / 阻碍主角？**（动机要合理）
4. **他 / 她的核心矛盾是什么？**（内心张力）

**任一问答不出，该配角不得出场。**

### 2.2 配角类型参考

| 类型 | 特征 | 经典参考 |
|---|---|---|
| **镜像角色** | 与主角相似困境，做出不同选择 | 《甄嬛传》华妃 vs 甄嬛 |
| **催化角色** | 推动主角成长，但自身不成长 | 《笑傲江湖》风清扬 |
| **对比角色** | 与主角形成鲜明对照 | 《天龙八部》萧峰 vs 段誉 vs 虚竹 |
| **成长型盟友** | 与主角一起成长，最终可能分道扬镳 | 《将夜》桑桑 |
| **亦师亦友型** | 同辈之间的引导者 | 《三体》罗辑 vs 章北海 |

### 2.3 配角雷区（避免）

- **工具人**：只在需要时出现，解决问题后消失
- **符号化**：只用一个标签代替性格（冷傲师姐、毒舌师兄、温婉女主）
- **没有独立时间线**：主角不在时，他们仿佛停止存在

### 2.4 性别比例（强制）

- 每引入 **3 名及以上新角色**，至少 **1 名为女性**
- 每卷结束后检查比例，明显失衡则下一卷补充
- 女性角色必须有独立欲望和成长弧光，**禁止**只做"花瓶"或"情感奖励"

### 2.5 身份表述

- 角色身份随剧情阶段动态调整（不要一直叫"杂役弟子"到结尾）
- 避免使用已过时的身份标签
- 关键转折点后，考虑切换对该角色的称呼方式

## 三、反派设计（四问）

### 3.1 强制四问

每个反派设计前，必须回答：

1. **他 / 她的终极目标是什么？**（必须是具体目标，不是"作恶")
2. **他 / 她自认为正义吗？**（最好 YES，悲剧性就出来了）
3. **他 / 她的弱点 / 软肋是什么？**（必须有）
4. **他 / 她的悲剧性在哪里？**（可恨之人必有可怜之处）

### 3.2 好反派示范

| 反派 | 终极目标 | 自认正义？ | 弱点 | 悲剧性 |
|---|---|---|---|---|---|
| 《天龙》慕容复 | 复兴大燕 | YES（家国立场） | 执念过深 | 被复国大业吞噬一切 |
| 《神雕》李莫愁 | 报复背叛过她的人 | YES（最初因爱生恨） | 对陆展元的执念 | 纯真少女变成杀人狂魔 |
| 《天龙》马夫人康敏 | 让所有看不起她的人臣服 | YES（她认为她理应被爱慕） | 自卑与虚荣 | 美貌是她唯一的武器，也是她最大的诅咒 |
| 《笑傲》岳不群 | 成为天下第一 | 曾经 YES | 野心反噬 | 被自己的虚伪吞噬 |
| 《人民的名义》祁同伟 | 跨越阶级 | 部分 YES | 太在意尊严 | 被出身碾压一辈子 |
| 《蝙蝠侠》小丑 | 证明所有人都会堕落 | YES | 没有真正的弱点（这本身是弱点） | 自己就是自己执念的囚徒 |
| 《基督山伯爵》 | 复仇 | YES | 对无辜者的伤害 | 复仇完成后发现自己一无所有 |

### 3.3 反派雷区（严禁）

- **弱智反派**：为了衬托主角而变蠢
- **纯粹作恶型**：无动机、无目标、只为恶而恶
- **开局登场既是 BOSS**：毫无铺垫直接让主角暴打
- **反派复读机**：每次登场都说"你不过如此"、"这不可能"

### 3.4 反派设计进阶

- **梯度反派**：小反派 → 中反派 → 大反派，每层都有独立动机
- **反派团体**：内部有分歧，不是铁板一块
- **反派背景**：为什么走上反派道路？有没有转向机会？
- **反派弧光**：反派也可以有成长（不一定是变好，也可以是更深的堕落）

## 四、角色关系网（强制）

### 4.1 主角关系矩阵

每卷至少维护以下类型的关系：

| 类型 | 必须 / 可选 | 说明 |
|---|---|---|
| 师长 / 长辈 | 必须 | 指引、训练、让主角仰视 |
| 同辈盟友 | 必须 ≥ 2 | 共同成长、互为支撑 |
| 同辈竞争者 | 必须 ≥ 1 | 激励主角变强 |
| 情感对象 | 可选 | 虐中带甜、克制隐忍 |
| 对立面 | 必须 ≥ 1 | 当前卷的主要对手 |
| 神秘角色 | 可选 | 身份未明，有长线钩子价值 |

### 4.2 金庸式背景落差（情感杠杆 · 模型7外延）

**核心原则**：情感张力来源于"不可能"。男女主角、朋友与敌人之间的背景差异越大，越能产生情感张力。

**强制运用**：
- 不要让关键关系"门当户对"。给情感关系设置"如果在一起就是背叛全世界"的外部障碍。
- **案例**：张无忌（明教教主/汉族）←→ 赵敏（元朝郡主/蒙古族）
- **案例**：令狐冲（正派大弟子）←→ 任盈盈（魔教圣姑）

**决策问题**：
- 这段关系中的"背景差异"是什么？够大吗？
- 如果在一起，代价是什么？这个代价是否让读者心疼？
- 关系中的张力是否来自外部阻力 + 内在吸引的双重作用？

### 4.3 关系动态变化

- 关系**不能静态**：每 5-10 章应有一次关系状态变化
- 关系变化应绑定**具体事件**（不是主角单方面"想通了"）
- 关系变化必须**记录到里程碑**（见 [milestone-rules.md](milestone-rules.md)）

## 五、角色设计自查清单

新角色引入前，逐项回答：

- [ ] （主角）三要素齐全？欲望、恐惧、矛盾都有吗？
- [ ] （主角）有没有专精外的难题？遇到会求援吗？
- [ ] （配角）四问都能答上吗？
- [ ] （配角）是工具人还是有独立时间线？
- [ ] （反派）四问都能答上吗？有自认正义吗？
- [ ] （反派）有可怜之处吗？还是纯粹作恶？
- [ ] 本卷新角色里女性比例达标吗？
- [ ] 角色称呼是否随剧情阶段调整？

任一项失败，**禁止该角色出场**，回到人物设计阶段。
