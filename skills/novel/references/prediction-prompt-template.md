# 故事涌现预测 Prompt 模板

> 本模板供 `story-builder` agent 在构造场景之前生成。由 `story-builder` 填充上下文后产出，交付给用户。用户可将此 prompt 提交给大模型，获取该场景下故事走向的涌现预测，作为故事构造的参考输入。
>
> 产出路径：`prediction-prompts/vol-{N}/ch-{X}-prediction-prompt.md`

---

## 使用说明

`story-builder` 按以下结构生成 prompt 文件后，用户有两种使用方式：

1. **独立使用**：把生成的文件内容复制给任意大模型，获取预测结果后反馈给 `story-builder` 继续构造
2. **内嵌调用**：若 `story-builder` 自身具备大模型调用能力，可直接以此 prompt 发起预测请求

---

```markdown
# vol-{N} · ch-{X} 故事涌现预测

> 编织者（story-builder）已为你准备了本章故事的全部上下文。请基于以下信息，**以角色的视角推演**故事的自然走向。

---

## 一、前情提要（最近 3 章摘要）

| 章节 | 核心事件 | 情绪收束 | 留下的悬念 |
|------|---------|---------|-----------|
| ch-{X-3} | ... | ... | ... |
| ch-{X-2} | ... | ... | ... |
| ch-{X-1} | ... | ... | **（本章开篇需承接此悬念）** |

## 二、涉及角色

> 以下是本章将要出场的角色及其完整画像。预测时请严格基于角色设定推演——**这个人会做什么，不是你想让他做什么**。

### 角色A：{姓名}（{身份}）
- **背景出身**：...
- **性格特质**：...
- **核心动机**：...
- **行为模式**：...（此人遇到冲突时的一般反应模式）
- **底线与禁忌**：...（此人在什么情况下会被触发、绝对不做的事）
- **当前状态**：...（上一章结束时的位置、情绪、身体状态）
- **当前目标**：...
- **与其他角色的关系**：
  - 与角色B：{关系类型}，当前状态：...

### 角色B：{姓名}（{身份}）
（同上结构）

### （如有更多角色，继续列出）

## 三、涉及地点

| 地点 | 描述 | 氛围 | 本章中的重要功能 |
|------|------|------|----------------|
| ... | ... | ... | ... |

## 四、预测任务

> 请针对以下命题，分别给出 2-3 个**符合角色性格和故事逻辑**的推演方向。
> 每个方向请包含：发生什么、为什么（角色动机驱动）、怎么发生的（场景简述）。

### 任务1：角色行为的自然推演

**命题**：在上一章收尾的悬念下，角色A 走进 {地点} 见到角色B，他最自然的第一个动作是什么？他的情绪会通过什么行为外显？

### 任务2：冲突升级的路径

**命题**：本章的核心冲突是"{冲突描述}"。在保持角色性格自洽的前提下，这场冲突有哪些合理的升级路径？哪条路径最能体现本章的情绪基调"{情绪基调}"？

### 任务3：钩子的推进方式

**命题**：本章需要推进以下钩子：{钩子列表}。在不刻意、不突兀的前提下，这些钩子的新线索通过什么方式自然露出最合适？

### 任务4：收尾悬念的设计

**命题**：本章的收尾应为后续章节留一个自然的悬念。基于本章的情绪曲线"{起→承→转→合}"，收尾悬念落在哪个角色、什么信息上最有力？

### （编织者可自由增删预测任务）

## 五、编织者期望

> 以下是我（story-builder）当前的想法。请以此作为参考基线，可以认同并深化、也可以有理有据地挑战——好的预测可能推翻我的预设。

- 本章我偏向的走向：...
- 我担心的风险点：...（例如"角色A 在此情境下可能显得被动，希望预测能给出让他主动破局的方向"）
- 我可能忽略的盲区：...（例如"配角C 的存在是否会对冲突产生我未预料到的影响？"）
- 期望预测提供的新视角：...

---

## 预测返回格式

请按以下结构返回你的预测结果：

```
### 角色行为推演
1. 方向一：...（为什么角色会这样做？基于他的什么性格/经历？）
2. 方向二：...
3. 方向三：...

### 冲突升级路径
1. 路径一：...（为什么这条路径更符合角色性格？）
2. 路径二：...
3. 路径三：...

### 钩子推进方案
1. 钩子"{钩子名}"：...（掉落方式 + 掉落时机）
2. ...

### 收尾悬念建议
1. 建议一：...（为什么这个悬念有效？对下一章的驱动力在哪？）
2. ...

### 对编织者期望的反馈
- 对当前走向的看法：...
- 是否发现了编织者未注意到的盲区：...
- 额外的灵感或建议：...
