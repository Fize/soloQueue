# 冲突设计（通用 · 金庸冲突根原则）

> 冲突是小说的发动机。没有冲突，就没有故事。本文件定义冲突的五种基本类型、升级曲线、明暗线搭配。
> **金庸模型4（隐藏的根）**：每部优秀的长篇都需要一个"埋藏的秘密/矛盾/谜题"驱动所有情节——冲突的"根"决定了故事的续航能力，根越深，故事能走越远。
> 打斗场景的冲突处理见 [combat-scene.md](combat-scene.md)；章节内冲突节奏见 [chapter-structure.md](chapter-structure.md)。

---

## 一、冲突的五种基本类型

| 类型 | 经典参考 | 特点 | 适合题材 |
|---|---|---|---|
| **人物 vs 命运** | 《悟空传》《俄狄浦斯王》 | 抗争宿命的悲壮感 | 仙侠、玄幻、神话 |
| **人物 vs 社会** | 《狂人日记》《变形记》 | 被体制碾压的无力感 | 官场、职场、架空历史 |
| **人物 vs 他人** | 《甄嬛传》《琅琊榜》 | 智谋博弈、情感纠葛 | 权谋、宫斗、江湖 |
| **人物 vs 自我** | 《罪与罚》《哈姆雷特》 | 内心挣扎、身份认同 | 所有题材通用 |
| **人物 vs 环境** | 《鲁滨逊漂流记》 | 绝境求生的坚韧 | 求生、探险、末世 |

### 1.1 冲突类型的搭配原则

- **每卷至少包含 2 种冲突类型**
- 主线冲突通常是前四种之一，**人物 vs 自我**应作为底色贯穿全书
- 不同 Arc 切换主冲突类型，避免读者审美疲劳

## 二、明线冲突与暗线冲突

### 2.1 明线冲突

- 读者**明确感知**的冲突
- 推动章节层面的情节
- **每章至少一条明线冲突**

**明线冲突示例**：

- 主角被挑衅 → 对决
- 主角任务未完成 → 限时完成
- 主角被陷害 → 洗清嫌疑
- 主角缺某资源 → 获取资源

### 2.2 暗线冲突

- 读者**部分感知**或尚未感知的冲突
- 铺垫长期矛盾，为后续大爆发蓄力
- **每 3-5 章埋一条暗线**

**暗线冲突示例**：

- 某个长辈的隐秘目的（等到爆发）
- 朝堂背后的势力博弈（等到牵连主角）
- 主角身世中的谜团（等到反转）
- 某个盟友的真实立场（等到背叛 / 反转）

### 2.3 明暗线交错原则

- 暗线**不能完全藏起来**：读者要能隐约感觉"有事"
- 每隔若干章应让暗线**浮现一下**（一句对话、一个眼神、一个不合常理的细节）
- 暗线浮现时不要点破，让读者自己嚼

## 三、冲突升级曲线（强制）

### 3.1 升级三段

冲突必须有升级曲线：**小冲突 → 中冲突 → 大冲突**

| 层级 | 典型跨度 | 表现 |
|---|---|---|
| 小冲突 | 单章 | 日常摩擦、局部危机 |
| 中冲突 | 3-5 章 Arc | 阶段性危机、较量 |
| 大冲突 | 卷级 | 卷终高潮、重大抉择 |

### 3.2 升级维度

冲突升级可以从以下维度之一（或多项）推进：

1. **对手变强**：同级对手 → 越级对手 → 最终 BOSS
2. **代价变高**：资源损失 → 朋友受伤 → 亲人生死 → 主角自己生死
3. **范围变大**：个人恩怨 → 门派之争 → 势力博弈 → 世界存亡
4. **认知变深**：表层矛盾 → 深层矛盾 → 世界真相
5. **道德困境加剧**：容易的选择 → 两难选择 → 必死选择

### 3.3 升级节奏

- **不能一直升级**：每次升级后要有喘息点（读者需要休息）
- **不能倒退**：升过的冲突层级不能下一卷又回到小冲突
- **不能跳层**：小冲突直接到大冲突会让读者跟不上
- **必须可视**：读者要能感受到"这次比上次更严重"

## 四、冲突根源分析（金庸"隐藏的根"模型）

### 4.1 冲突必有"根"（金庸模型4）

**核心原则**：每部优秀的长篇小说都需要一个"隐藏的根"——深埋在故事之下的秘密、矛盾或谜题，它不直接出现，但驱动所有情节。

**金庸案例**：
- 《射雕》的根：《九阴真经》——所有人都在争夺的武功秘笈
- 《天龙》的根：慕容氏复国阴谋 + 萧远山30年前的伏击——两个"过去的事件"驱动所有命运
- 《笑傲》的根：任我行失位、左冷禅图谋并派——权力真空引发的江湖震荡
- 《倚天》的根：成昆因情生恨——私人情感驱动整个江湖的腥风血雨

**应用**：在大纲阶段就问自己：这个故事之下埋着什么？是什么"过去的事件"或"隐藏的矛盾"在驱动现在发生的一切？

**设计方法**：
- 这个"根"必须在前中期就开始埋，后期再揭露
- 如果一开始就在面上，它就只是普通的"主线"而非"隐藏的根"
- 根越深，故事能走的距离越远

## 五、冲突的推进手法

### 5.1 推进方式

| 方式 | 适用场景 |
|---|---|
| **直接对抗** | 战斗、辩论、谈判 |
| **信息战** | 误会、情报、揭露 |
| **利益博弈** | 联盟、交易、出卖 |
| **心理战** | 试探、诱导、反间 |
| **时间压力** | 倒计时、限期完成 |
| **三方介入** | 第三方打破平衡 |

### 5.2 避免冲突平庸化

- **避免一对一死磕**：加入第三方变量
- **避免信息对称**：让读者知道一些人物不知道的事（产生紧张感）
- **避免输赢立见**：让胜负悬在高空（长期拉锯）

## 六、冲突设计自查清单

设计一条冲突线时，逐项检查：

- [ ] 这条冲突属于五种基本类型中的哪种？
- [ ] 它是明线还是暗线？
- [ ] 它的升级曲线是什么？小 → 中 → 大 能画出来吗？
- [ ] 它的根源是什么？双方真实诉求是什么？
- [ ] 它是"可调和"还是"不可调和"？
- [ ] 它在哪一章爆发？哪一章回收？
- [ ] 本卷有没有至少 2 种冲突类型？
- [ ] 本章的明线冲突有没有推进？（不能原地踏步）

任一项答不上，**说明冲突设计尚未完成**，回去补。
