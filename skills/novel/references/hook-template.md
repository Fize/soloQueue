# 剧情钩子模板 (渐进式披露规范)

> 本文件是剧情钩子的标准模板。长线与中线钩子必须建立独立的追踪文档，并遵循本模板结构。

---

```markdown
---
name:         hook-{简短名称}
type:         plot_hook
level:        short_term / mid_term / long_term (短线 1-3 章 / 中线 5-15 章 / 长线数十章)
status:       Seed / Active / Revealing / Resolved
created_at:   ch-{引入章号}
resolved_at:  est-ch-{预计回收章号}
---

## 1. 读者已知线索 (Public)

> 【对正文创作可见】本节仅以列表形式记录当前及历史章节中已正式在正文里掉落的物理事实、线索或主角已知信息。
- **ch-X**：{已掉落的线索描述}
- **ch-Y**：{已推进的内容描述}

## 2. 创作者大纲占位 (Hidden)

> 【对正文创作屏蔽，仅在生成章节蓝图时作为编排参考】本节记录未来的大纲走向，避免 AI 在写作时由于阅读了本节内容而产生“全知视角”剧透。
- **最终回收设计**：{简述本钩子最终将如何收尾揭底}
- **阶段线索掉落计划**：
  - **ch-A (预估)**：{计划在这一章掉落的下一条线索/变量}
  - **ch-B (预估)**：{计划如何再次推进}

## 3. 关联人物与势力

- **涉及角色**：[{角色A}](file:///../人物体系/{角色A}.md)、[{角色B}](file:///../人物体系/{角色B}.md)
- **涉及势力**：{宗门或势力名称}
```
