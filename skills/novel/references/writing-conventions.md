# 创作节奏与语言规范（通用 · 修玄基线）

> 本文件定义全书规模、字数配比、语言规范（时间 / 度量 / 修玄身份言行）的**默认基线**。
> 默认适配修真 / 玄幻 / 仙侠题材。现代 / 都市 / 科幻题材可通过项目红线外挂文件覆盖。

---

## 一、全书规模与节奏

### 1.1 基准规模（可由项目覆盖）

| 维度 | 基准值 |
|---|---|
| 全书卷数 | 10-14 卷 |
| 全书章数 | 500-1000 章 |
| 全书字数 | 200-300 万字 |
| 每卷字数 | 15-24 万字 |
| 每卷章数 | 50-80 章 |
| 每章字数 | 3000 字左右（核心战役 / 高潮章节可至 5000） |

### 1.2 每卷结构节奏

| 阶段 | 章节占比 | 主要任务 |
|---|---|---|
| 开卷凤头 | 0-10% | 承接上卷悬念，埋新卷主线 |
| 铺展 Arc 1 | 10-30% | 主角进入新环境 / 新关系 |
| 推进 Arc 2 | 30-55% | 第一次小高潮，暗线浮现 |
| 危机 Arc 3 | 55-80% | 冲突升级，中高潮 |
| 卷末 Arc 4 | 80-95% | 大高潮，真相部分揭露 |
| 卷尾 | 95-100% | 悬念收束 + 下一卷钩子 |

### 1.3 核心爽点密度

- **每卷**：3-5 个核心爽点
- **每 3 卷**：至少一个大高潮
- **每章**：2-3 个小爽点（红线）

### 1.4 节奏双驱动

小说节奏由两个变量驱动：

1. **换地图**：主角从一个场景切换到下一个（灵脉 → 宗门 → 秘境 → 异界）
2. **换境界 / 阶段**：主角能力层级跃升（入门 → 中期 → 后期 → 突破）

**节奏法则**：每 1-2 卷完成一次地图切换 + 境界跃升，避免在同一格局下停留过久。

### 1.5 钩子嵌套节奏

- **长线**（贯穿全书）：1-3 条
- **中线**（跨 3-5 卷）：3-5 条
- **短线**（卷内回收）：每卷 2-4 条

详见 [chapter-structure.md](chapter-structure.md) §2.1。

## 二、语言规范（修玄题材默认）

### 2.1 禁用现代词汇

| ❌ 现代词 | ✅ 修玄替代 |
|---|---|
| 秒、分钟、小时 | 瞬息、弹指、一炷香、半个时辰、一个时辰 |
| 毫米、厘米、米 | 几寸、数尺、丈许 |
| 克、千克、吨 | 几斤、数两、若干石 |
| 计算机、数据库、网络 | 不应出现 |
| 订单、客户、业绩、KPI | 不应出现 |
| 营业执照、税务 | 不应出现 |
| 信号、系统 | 不应出现 |
| 效率、流程、团队 | 改成火候、章程、同门 |
| 数据、概率 | 改成天数、几率（慎用）、造化 |
| 瞬间 | 可用，但在正式场合改为"刹那"或"电光火石" |

### 2.2 推荐时间单位

- **瞬息**：一个呼吸时间之内
- **弹指**：约数秒到十余秒
- **一炷香**：约半刻钟到一刻钟
- **半个时辰** / **一个时辰**：半小时 / 一小时（但不要说"小时"）
- **一日 / 一夜 / 一月 / 一年**
- **百年 / 千载**：修士寿元单位

### 2.3 推荐度量单位

- **几寸 / 尺许 / 丈许 / 数丈 / 百丈 / 千丈 / 里许 / 百里**
- **几斤 / 数两 / 石**（重量）
- **盏 / 坛 / 瓮**（容器）
- **缕 / 丝 / 片 / 缕缕**（灵力 / 气息）

### 2.4 风格一致性

- **所有文档**（正文、蓝图、里程碑、钩子、大纲）统一使用题材语言
- 策划文档可以使用部分现代词（如"配比"、"结构"），但**正文绝对不允许**
- 禁止武侠题材标签混入修真题材（如"江湖侠客"跑进修真世界）

## 三、修玄身份言行规范（红线）

> 与 [anti-ai-patterns.md](anti-ai-patterns.md) §六协同执行。本文件侧重"节奏与基线"，anti-ai-patterns 侧重"红线检查"。

### 3.1 角色身份与言行一致性

- 修真者言行必须符合其**修为境界和身份地位**
- 低阶对高阶必须体现**敬畏**（称谓：前辈、真人、道友等）
- 同阶相称符合修真界惯例（道友、师兄、师弟）
- **严禁**修真者使用凡人市井语言或现代口语

### 3.2 场景设定修真化

- 修真场景必须符合修真世界观（洞府、灵脉、宗门、秘境）
- 禁止出现凡人世界元素（现代建筑、科技产物、凡俗商业）
- 日常生活修真化（灵石交易、传音符通讯、飞剑代步）

### 3.3 凡人 vs 修士的区别

- 修真者寿元、眼界、格局远超凡人
- 即使在凡人界行走，本质仍是修真者思维
- 不得以凡人逻辑解决修真问题（"讲道理"代替实力说话）

## 四、段落与移动端适配

### 4.1 段落长度（红线）

- 目标：**3-5 行**
- 警告：**6-8 行**（黄线）
- 禁止：**超过 8 行**（红线 FAIL）
- 对话：单行或双行换段

### 4.2 移动端阅读节奏

- 每章 2-3 个情绪释放点
- 每 500 字至少一次场景 / 对话切换（避免连续叙述压得人喘不过气）
- 避免连续 3 段以上纯环境描写

## 五、自查清单

每章写完后逐项检查：

- [ ] 本章字数在目标范围内？（约 3000，极端场景可至 5000）
- [ ] 段落没有超过 8 行的？
- [ ] 现代词汇（§2.1 禁用表）有没有泄漏到正文？
- [ ] 时间 / 度量单位是否已修玄化？
- [ ] 修真者对话是否符合身份境界？
- [ ] 场景描述是否纯修真化？

任一失败 → 回到章节创作分支 `story-engine` 修订。
