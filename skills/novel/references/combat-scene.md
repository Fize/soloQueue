# 打斗场面规范（通用 · 修真/玄幻基线）

> 本文件定义打斗场景的硬性要求，默认适配**修真 / 玄幻 / 仙侠**题材。武侠 / 都市格斗 / 科幻战斗等题材可通过项目红线外挂文件覆盖。
> 本文件与 [writing-style.md](writing-style.md) §五（对话情绪）、[anti-ai-patterns.md](anti-ai-patterns.md) §三（五感）、[chapter-structure.md](chapter-structure.md)（章节结构）联动使用。

## 一、核心原则

**打斗场面必须严格遵循境界设定，体现不同境界之间的实力差异**，同时兼顾节奏、五感、智慧博弈。

- **境界差异展示是红线**：不同境界之间必须拉开可视差距
- **五感 ≥ 3** 是红线
- **智慧博弈层次**是黄线（要有，但不阻断交付）

---

## 二、境界差异的通用模型

> 具体境界名称由项目定义（项目红线文件中声明），本文件只给通用规则。

### 2.1 通用境界差距规律

| 差距 | 表现 |
|---|---|
| 同境界 | 胜负在功法、法宝、经验、地形、心境 |
| 差半境 | 低境界需要极大优势（秘术、特殊法宝、地形）才能抗衡 |
| 差一境 | 碾压级差距，低境界以逃为主、以智为辅 |
| 差两境及以上 | 神识一扫 / 一指即灭；低境界只能找缝隙、借势、不能硬拼 |

### 2.2 打斗中"展示境界差"的惯用手法

- **对手法术轻松破防**：低境界倾尽全力的一击被高境界随手化解
- **速度跟不上**：低境界连对方出手轨迹都看不清
- **真元量差距**：低境界拼消耗必败
- **神识压制**：高境界一扫，低境界心神欲裂
- **气势压制**：高境界尚未出手，低境界已经喘不过气
- **法术余波伤人**：高境界对他人交手，余波就足以让低境界重伤

### 2.3 越级挑战的合理设定

> **越级 ≠ 开挂**。越级必须有合理支撑，否则违反境界规则。

合理支撑候选：

- 特殊功法（能借力、能伤敌于无形）
- 强大法宝（等级高于修为的法宝）
- 秘宝加持（如体内某件本源之物）
- 对方受伤 / 轻敌 / 被限制
- 地形利用 / 阵法辅助
- 同伴配合 / 多打一

**越级挑战必须付出代价**：真元透支、反噬、伤势、寿元损耗、后遗症。

## 三、打斗节奏

### 3.1 不同层级打斗的节奏

| 层级 | 典型节奏 | 威力范围 | 战斗时长 |
|---|---|---|---|
| **低阶**（入门级） | 近身为主，法术辅助，快速短促 | 方丈 - 数丈 | 数息 - 半刻 |
| **中阶** | 远近结合，御空游斗 | 数丈 - 百丈 | 一刻 - 小半个时辰 |
| **高阶** | 大场面法术，天地异象 | 百丈 - 十余里 | 较长，节奏多变 |
| **顶阶** | 触发道韵，规则级压制 | 里级以上 | 长短取决于胜负判定 |

### 3.2 节奏控制原则

- **高潮层层递进**：不是一开场就全力，而是试探 → 交锋 → 真章
- **节奏要有呼吸**：紧凑战斗之间穿插观察 / 对话 / 计算
- **结局要有余波**：胜负落定后必须有一段"余音"描写（气血翻涌、灵气紊乱、环境回复）

### 3.3 打斗中的信息拉扯

- 打斗过程中让读者**不断获得新信息**（对方底牌、主角新领悟、环境异变）
- 避免"一整段只写招数名字"
- 每一次交锋应**改变战场态势**（优劣势转换、资源消耗、心境波动）

## 四、五感描写（红线 · ≥ 3 种感官）

打斗是关键场景，**强制五感 ≥ 3 种**。常用搭配：

| 感官 | 打斗中的典型表现 |
|---|---|
| 视觉 | 法术光芒、飞剑轨迹、身法残影、血花 |
| 听觉 | 法术爆鸣、金铁交击、破空风声、闷哼 |
| 触觉 | 灵力冲击、气浪余波、地面震动、冰冷 / 灼热 |
| 嗅觉 | 血腥气、烧焦味、灵药香、臭氧味 |
| 味觉 | 口中铁锈味、苦涩、血腥 |
| 玄幻特有 | 灵压、神识冲击、道韵、因果感知、气运紊乱 |

**常见错误**：

- 整段只有视觉：加入听觉（破空声）+ 触觉（气浪）
- 感官堆砌成流水账：让五感服务于情绪变化，不是罗列
- 高潮处反而没有五感：关键一击必须至少 3 种感官同时到位

## 五、实力差距的 Show 而非 Tell

> 这是打斗场景最容易翻车的点——直接告诉读者"他很强"，读者不信。

### 5.1 错误 vs 正确

| ❌ Tell | ✅ Show |
|---|---|
| 他很强 | 他随手一指，十丈外的青石崩成粉末 |
| 对方实力远胜于他 | 他拼尽全力的一剑，只在对方指尖留下一道白印 |
| 气势压人 | 小师妹脸色煞白，手中长剑"当啷"落地 |
| 神识强大 | 那道神识扫过时，主角只觉得整个人被倒提在空中，连心跳都被看穿 |
| 战斗很激烈 | 洞府顶上的石钟乳一节节折断，落下来砸在地面上又被余波碾成粉 |

### 5.2 常用 Show 手法

1. **用环境反应展示威力**：地面崩、空气燃、水流倒卷
2. **用旁观者反应展示实力**：围观者的脸色、惊呼、后退、屏息
3. **用数据对比展示差距**：主角全力一击的轨迹 vs 对方随意一指的轨迹
4. **用时间错位展示速度**：主角看见出手 → 身体已经挨了 → 眼睛才反应过来

## 六、打斗中的智慧博弈（黄线 · 猫腻风格强化）

> 猫腻风格的打斗不是单纯力量比拼，而是**智慧与经验的较量**。

### 6.1 战术五层次

| 层次 | 手法 |
|---|---|
| 第一层 | 硬碰硬，实力对拼 |
| 第二层 | 功法相克，法宝制衡 |
| 第三层 | 地形利用，环境变化 |
| 第四层 | 心理博弈，预判与反预判 |
| 第五层 | 局外布局，以打斗为诱饵 |

### 6.2 主角打斗时的智慧要求

- 体现**冷静分析、临场应变、策略调整**
- 用**智谋弥补实力差距**，用**战术取得优势**
- 打斗中可穿插**心理战、信息战、环境利用**

### 6.3 不是每场打斗都要五层

- **低阶小战**：第一 / 二层就够
- **中阶关键战**：至少第三层
- **卷级高潮战**：建议到第四 / 五层
- **BOSS 对决**：第五层（局外布局）最能打动读者

## 七、打斗与境界突破

### 7.1 突破时机

打斗是境界突破的**催化剂**：

- 生死危机中突破瓶颈
- 与高手过招中领悟道韵
- 情感剧变中心境蜕变

### 7.2 突破必须合理

- **有前兆**：近几章铺垫"马上要突破了"
- **有积累**：前期修炼、机缘、心境修持
- **有契机**：某个具体事件触发

**禁止**：

- 打着打着突然突破（无铺垫）
- 一突破就无敌（境界瞬间跳级，破坏设定）
- 突破后立刻大杀四方（至少需要磨合期）

## 八、打斗场景自查清单（红线 + 黄线）

### 8.1 红线项（必须 YES，否则 FAIL）

- [ ] 境界差异是否充分展示？（通过环境、反应、手法）
- [ ] 实力差距是**展示**还是**说明**？（有 Tell 就红线失败）
- [ ] 五感描写是否 ≥ 3 种？
- [ ] 越级挑战是否有合理支撑？
- [ ] 打斗结果是否合乎境界逻辑？

### 8.2 黄线项（建议 YES，不过不阻断交付）

- [ ] 打斗节奏是否符合境界层级？（低阶短促、高阶大场面）
- [ ] 主角是否体现了智慧博弈？（至少到第二层）
- [ ] 打斗中是否有信息拉扯？（获得新底牌 / 领悟）
- [ ] 胜负落定后是否有余音描写？
- [ ] 如果发生突破，前兆 + 积累 + 契机是否齐全？

## 九、打斗场景常见翻车点

### 9.1 境界混乱

- 同一个高境界角色，上一章对主角碾压，下一章被主角一剑斩杀 → **红线 FAIL**
- 设定里不可能做到的事，为了剧情强行做到 → **红线 FAIL**

### 9.2 一拳超人式

- 主角不管对手什么境界，一拳 / 一剑解决 → 读者无代入感
- 解决方法：让胜利有**代价**（伤势、反噬、资源消耗）

### 9.3 PPT 打斗

- 只写招数名字，不写对抗细节
- 解决方法：每个招数落下，必须有**环境反应 + 对手反应 + 主角内心**

### 9.4 情绪抽离

- 打得热闹，但主角不害怕、不紧张、不愤怒、不心疼
- 解决方法：让战斗服务于情绪，不是独立演武
