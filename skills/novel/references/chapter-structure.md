# 章节结构与钩子设计（通用）

> 本文件定义单章内部结构（凤头/猪肚/豹尾）与长短钩子的设计规则，是章节创作分支 `story-engine` 写蓝图与正文的主要依据。
> 文风相关见 [writing-style.md](writing-style.md)；质量红黄线见 [quality-checklist.md](quality-checklist.md)。

## 一、章节叙事法度与节奏（起承转合 · 金庸技法联动）

> 本章节结构与 `jin-yong-writing-perspective` 的技法3（忙中作闲）、技法7（章回钩子）、技法10（闲笔蓄势）联动执行。

作为单章输出，必须具备完整的叙事弧线，拒绝平铺直叙。每个章节必须严格按照"起、承、转、合"的法度推进：

### 1.1 起（环境与心境切入 / 凤头）
- **切入法度**：章节开头不直入主题，先用 2-3 句白描铺陈当前环境（天象、节气、地理）或人物内心的气机流转，定下本章的情感基调。
- **顺应叙事氛围**：可根据当前情节需要，以动作与冲突切入，或以诗意、安静的环境渲染开局。

### 1.2 承（细节与张力酝酿 / 猪肚前段）
- **张力酝酿**：通过对话、周遭人物的反应、或是极其细微的环境变化（如：茶杯中的水泛起涟漪、风向骤变），积攒冲突爆发前的张力。
- **实质性推进**：事件必须产生实质进展，且伴随着角色心理状态、形势的有机变化。

### 1.3 转（核心冲突/斗法 / 猪肚后段）
- **武戏文唱**：战斗情节必须拆解为“引气 - 出招/施法 - 轨迹/异象 - 碰撞反馈”。
- **节奏把控**：多用动词短句，加快阅读节奏。招式名称需契合中国传统哲学（易经、佛道）。不再强求死板的爽点数量，顺应逻辑爆发。

### 1.4 合（余韵与悬念 / 豹尾）
- **留白与钩子**：章节结尾绝不收得太满。用一个意想不到的变故、一句暗藏玄机的台词、或一个悠远深邃的空镜头（如：“大雪未停，那具残尸已被掩埋了一半”）作为结束，留足悬念。
- **禁止空洞套话**：杜绝如“一切才刚刚开始”、“这一夜，注定不平静”等无意义的陈词滥调。

## 二、钩子体系设计

### 2.1 钩子三层嵌套

| 层级 | 跨度 | 典型内容 | 回收位置 |
|---|---|---|---|
| **短线钩子** | 章内 / 1-3 章 | 悬念小事件、局部疑点 | 章末或下一章 |
| **中线钩子** | 5-15 章 | 阶段性目标、身份疑点 | 卷内或跨卷初 |
| **长线钩子** | 数十章到全书 | 身世之谜、终极目标、世界真相 | 全书中后段 |



### 2.3 长线钩子管理（红线）

- 每个**跨越 5 章以上**的钩子**必须**有追踪文档
- 路径：`plots/active-hooks/hook-{名称}.md`
- 必须包含字段：
  - **引入章节**：什么时候埋下
  - **核心悬念**：这个钩子到底在钩读者什么
  - **线索掉落计划**：每隔几章掉一块新线索
  - **回收节点**：哪一章揭底
  - **当前状态**：未引入 / 已引入 / 推进中 / 待回收 / 已回收

### 2.4 五章节点钩子（红线 · 易漏项）

> **最容易被遗漏的红线**：每逢第 5、10、15、20…… 章这类五章节点，**必须明确挂出一个"下一阶段钩子"**，不能只做收束、不做引出。

**强制要求**：

- 五章节点章节必须让读者清晰感到：这一小段剧情已经落锤，但**更大的麻烦、目标、秘密、追杀、去处或选择已经逼到眼前**
- 所谓"下一阶段钩子"，必须从以下维度中**至少占一项**：
  1. 新目标出现（主角主动要去追什么）
  2. 旧谜团加深（已有疑点又多出一层）
  3. 敌方动作升级（对手开始出牌或换人）
  4. 关键关系变化（盟友 / 仇敌关系翻转或深化）
  5. 关键资源 / 地点 / 人物线索出现（新地图、新人物、新物件）

**违规表现**（红线 FAIL）：

- 第 5 章只总结前 5 章内容，没有引出新的麻烦
- 第 10 章里程碑里只有"阶段成果"，没有"下一阶段钩子"
- 豹尾的悬念和五章节点无关，读者不知道"接下来要追什么"

### 2.5 钩子设计四大原则

1. **明确回收节点**：每个钩子必须有回收计划，不能埋了就忘
2. **钩子间有内在联系**：不是孤立事件，要编织成网
3. **层次感**：短线、中线、长线嵌套，让读者既有短期爽感也有长期期待
4. **回收时的爆发感**：回收钩子必须有"原来如此"的快感，不是平淡揭底

## 四结构自查清单

章节写完后，进行自然审视：

- [ ] 开篇：叙事切入点是否契合当前的叙事氛围和角色状态？
- [ ] 展开：情节或人物关系是否产生了实质的、符合逻辑的推进？
- [ ] 收尾：收尾是自然的还是刻意的？是否规避了空洞的总结套话？
- [ ] 本章是 5 章节点吗？如果是，挂出下一阶段钩子了吗？（按 milestone 规范）
- [ ] 本章用到的活跃钩子有没有推进？
- [ ] 本章引入的新钩子（跨 5 章以上）有没有建追踪文档？

任意一项失败，**禁止交付**。
